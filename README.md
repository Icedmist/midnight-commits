# Midnight Attendance 🌙

> **Offline Logger v1.0** — A lightweight, offline-first attendance tracking system built with Streamlit and SQLite.

## Features

- **Interactive Selection**: Easily mark student status (Present ✅, Absent ❌, Late ⏰).
- **SQLite Database Persistence**: Securely stores records locally in `attendance.db`.
- **Live Logs Dashboard**: Real-time admin table view of logged entries.
- **CSV Data Export**: Instant data extraction for offline reporting.

## Setup & Running

```bash
# Install dependencies
pip install streamlit pandas

# Run the application
streamlit run app.py
```
